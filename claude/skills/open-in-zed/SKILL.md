# open-in-zed

指定されたパスをZedエディタで開く。

## 手順

1. ユーザーのメッセージからパス（ファイルまたはディレクトリ）を抽出する
2. Zedがインストールされているか確認する)
```bash
type -p zed > /dev/null 2>&1
```
3. Zedでユーザーが指定したパスのファイルを開く
4. 実行後、「`<path>` をZedで開きました」と報告する

## 注意

- 相対パスが渡された場合は絶対パスに解決してから渡す
- パスが存在しない場合はその旨をユーザーに伝える（Zedは起動しない）
